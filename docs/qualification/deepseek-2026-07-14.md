## Deterministic implementation result

- The package produces one explicit `pho` binary. Cargo automatic binary discovery is disabled, so the frozen `phase1_probe` and ChatGPT modules remain on disk but outside the ordinary compile graph.
- `pho login`, `status`, `logout`, `chat`, and explicit `chat --stdin` dispatch shared intents through one reducer and coordinator. Prompt/key arguments and implicit piped prompt mode are rejected.
- The DeepSeek credential actor transactionally validates and replaces a distinct versioned Keychain record, preserves a prior record after candidate validation failure, gates leases, and reports deletion failure without claiming logout.
- The concrete backend builds a bounded fixed-origin request, maps a bounded SSE stream into authoritative assistant phases, retains usage, and performs no automatic model retry.
- The qualification-only in-memory `phase1b_echo` tool strictly validates complete arguments, executes calls sequentially, preserves provider/local identities, and replays the complete reasoning-bearing assistant phase.
- Process tests cover help/version, invalid argv, no-TTY login, explicit stdin selection, missing credentials, and seeded-marker redaction. SIGINT and renderer failure are wired to coordinator cancellation, but require manual process evidence.
