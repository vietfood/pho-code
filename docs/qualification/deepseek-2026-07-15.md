## Pricing and the $5 balance

The official [DeepSeek pricing page](https://api-docs.deepseek.com/quick_start/pricing/) reviewed on 2026-07-15 lists `deepseek-v4-flash` at USD $0.0028 per 1M cache-hit input tokens, $0.14 per 1M cache-miss input tokens, and $0.28 per 1M output tokens. Prices are volatile; the provider ledger remains authoritative.

Ignoring mixed traffic and integer rounding, a $5 balance corresponds to approximately:

- 1,785,714,285 cache-hit input tokens;
- 35,714,285 cache-miss input tokens; or
- 17,857,142 output tokens.

For mixed traffic, Pho Code estimates:

```text
USD = cache_hit_tokens * 0.0028 / 1,000,000
    + cache_miss_tokens * 0.14 / 1,000,000
    + completion_tokens * 0.28 / 1,000,000
```

Reasoning tokens are included in completion tokens and are not charged twice by the estimator. The observed 37-token command sample cost about 0.0001764% of the $5 balance. The balance is ample for qualification and ordinary harness development, but Pho Code does not know the remaining provider balance and does not enforce a monetary cap.
